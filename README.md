【游戏名称】
户外滑雪

【游戏介绍】
这是一款H5原生开发使用 Three.js 3D库 和 FBX 3D 模型的休闲滑雪类小游戏。游戏中，你扮演一名滑雪者，在雪地上快速滑行。你需要左右移动躲避前方的树木和石头障碍，滑得越远得分越高！随着游戏进行，速度会越来越快，挑战你的反应能力。

【玩法说明】
游戏中，你的滑雪者会自动向前滑行，雪地上会不断出现树木和石头障碍物。

障碍：
1. 树木和石头会从屏幕下方不断出现，向上移动。
2. 如果撞到任何障碍物，游戏就会结束。
3. 随着得分增加，滑行速度会越来越快，障碍物出现频率也会提高。

操作：
点击屏幕左侧按钮（或按左方向键）- 向左移动
点击屏幕右侧按钮（或按右方向键）- 向右移动
按回车键 - 游戏结束后重新开始

目标：
躲避所有障碍物，尽可能滑得更远，获得更高的分数！

Tips：游戏会自动记录你的得分。
速度越快，得分增长越快。提前预判障碍物位置，保持冷静，挑战最高分！

【技术介绍】
本游戏使用 Three.js WebGL 3D渲染引擎开发，结合以下技术栈：

核心技术：
- Three.js (v92) - WebGL 3D图形库，负责场景渲染和3D对象管理
- FBX 3D模型 - 使用FBXLoader加载带骨骼动画的3D角色和物体模型
- HTML5 Canvas - 作为WebGL渲染目标
- JavaScript ES6 - 游戏逻辑编程语言

渲染技术：
- WebGL - 硬件加速的3D图形渲染
- 纹理贴图 - 使用JPG图片作为雪地纹理，通过TextureLoader加载
- 材质系统 - MeshLambertMaterial（兰伯特材质）支持光照效果
- 光照系统 - 环境光(AmbientLight) + 半球光(HemisphereLight)
- 雾效(Fog) - 营造远景深度感

3D模型与动画：
- 角色模型 - player@skateboarding.fbx（带滑板动作的骨骼动画）
- 滑雪板模型 - snowboard.fbx（附加到角色脚部骨骼）
- 树木模型 - tree.fbx（通过克隆创建多个实例）
- AnimationMixer - Three.js动画混合器处理骨骼动画播放

几何体与场景：
- BoxGeometry - 创建地面、山坡等基础几何体
- 动态场景生成 - 随机生成平台和障碍物
- 碰撞检测 - 使用Box3包围盒进行碰撞判定
- 相机系统 - PerspectiveCamera（透视相机）+ 第三人称跟随

性能优化：
- 对象池 - 树木模型克隆复用
- 视锥剔除 - 移除屏幕外的对象
- LOD管理 - 根据距离动态加载/卸载对象

所有图形都在Canvas上通过WebGL实时渲染，实现流畅的3D滑雪体验。
